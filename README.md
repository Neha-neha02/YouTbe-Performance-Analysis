# YouTube Performance Analysis Dashboard

## 📊 Project Overview
This project analyzes YouTube video data from 2000–2021 to understand video performance, audience engagement, and trends using Tableau. The dashboard provides interactive visualizations to explore top-performing videos, channels, and viewer interactions.

---

## 🔍 Key Insights
- **Top 10 Most Viewed Videos:** Identify the videos that attracted the most audience attention.  
- **Top Performing Channels:** Highlight channels with the highest total views.  
- **Views vs Likes:** Shows the relationship between views and likes for each video to understand engagement.  
- **Views vs Comments:** Indicates audience interaction levels on each video.  
- **Channel Contribution by Views:** Pie chart showing each channel's contribution to total views.  
- **Video Upload Trend Over Time:** Line chart reflecting content growth on YouTube across years.  

---

## 🛠 Tools Used
- **Tableau Desktop** – For building interactive dashboards and visualizations.  
- **YouTube CSV Dataset** – Contains channel names, video titles, published dates, views, likes, and comments.  

---

## 📁 Dataset Description
The dataset contains the following columns:  

| Column Name       | Description |
|------------------|------------|
| Channel Name      | Name of the YouTube channel |
| Published Date    | Date the video was uploaded |
| Title             | Video title |
| Views             | Total views |
| Likes             | Total likes |
| Comments          | Total comments |
